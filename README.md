# hola-mudo-java

##El programa "Hola mundo" simplemente muestra "Hola mundo" por pantalla

## Compilacion y ejecucion del programa

para **compilar** el programa teclea lo siguiente (Nota: Es necesario 
tener el "JDK"):

---Console
javac HolaMundo.java
--- 

y para **Ejecutarlo**:

---Console
java HolaMundo
---
